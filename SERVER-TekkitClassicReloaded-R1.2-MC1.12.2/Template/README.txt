------------------------------------------------------------
 WINDOWS
----------
 Use one of the following options:

 1. Double click "ServerStart.bat" (recommended).
 2. Double click "forge-1.12.2-14.23.5.2847-universal.jar".


------------------------------------------------------------
 LINUX
----------
 To start on any general Linux distribution like Debian, CentOS or Ubuntu
 you will have to run the shell script "ServerStart.sh". Navigate to the
 server directory using your preferred SSH terminal and enter the command:

 bash ServerStart.sh
